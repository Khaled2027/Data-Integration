This is the final project for the B142 Data Integration course
# Instructions for Loading the Data (Databricks Setup)

## 1. Import the DB File

- Download the `.dbc` file
- Go to Databricks
- Import the `.dbc` file into your workspace

## 2. Open the Notebook

- After importing, open the workspace
- Navigate to the notebook called: **Data Exploration and Data**

## 3. Download Dataset from Google Drive

- Open the link below:
  [Google Drive Dataset](https://drive.google.com/drive/folders/1iBKZog_PI4TLsDHhbhiHAoIzBv_D-1NC?usp=share_link)
- Download the ZIP file
- Uncompress (extract) the files

## 4. Upload Data to Databricks Volume

- Go back to the Part 1 notebook
- Run the first SQL cell to create:
  - catalog
  - schema
  - volume

## 5. Upload Files to Volume

- Go to the Databricks Workspace
- Refresh the page
- Click on the created volume
- Select **"Upload to this Volume"**
- Upload the extracted files
